# SMT-Barcode
Web App That Using a QR code scan to saves the time you arrive at the production line

##### The app is fully responsive and its main use is from mobiles.

## Programming Languages
#### Server Side:
* Node.js
* Express.js
* MongoDB
#### Client Side:
* React.js

#### JWT Authentication
